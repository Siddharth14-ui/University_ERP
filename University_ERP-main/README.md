# University ERP System

## Features
- Student Management (Add/Delete)
- Course Management
- Fee Management
- Payment Tracking
- Attendance System

## Tech Stack
- Java (Swing)
- JDBC
- MySQL

## How to Run
1. Setup MySQL database
2. Update DB credentials in DBUtil.java
3. Run FeeReportUI.java

## Database Setup

1. Create DB:
   CREATE DATABASE erp_system;

2. Import file:
   SOURCE erp_system.sql;
